 Project Requirement Document (PRD)
Project Name: 
CloudDrive
 (Mini Cloud Storage System)
1. 
📌
 Project Overview
CloudDrive
 is a web-based cloud storage application that allows users to upload, store, manage, and share files online. The system is inspired by platforms like Google Drive but is designed with a simplified scope suitable for a mid-semester MCA project.
The application provides secure authentication, file management, folder organization, and cloud-based storage using AWS services.
2. 
🎯
 Objectives
Build a full-stack cloud storage system using Java backend and React frontend
Implement real-world cloud architecture using AWS
Enable users to manage files (upload, download, delete, share)
Demonstrate understanding of REST APIs, authentication, and cloud deployment
